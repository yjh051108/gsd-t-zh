## 这是什么

此片段定义了 **声明声明** 要求，注入到每个符合条件的 GSD-T 阶段提示词中（plan、pre-mortem、partition、discuss、milestone [D3 — 上层阶段]；execute、debug、quick [D4 — worker 阶段]）。

**DETECT 不是确定性的。** 检测你需要信息是一种判断。此片段是一个 LLM 提示义务（尽力而为），镜像 keep-or-supersede 协议。确定性存在于 CLASSIFY（`bin/gsd-t-research-gate.cjs`）和 ENFORCE（§7 标记 + 验证门控）中 — 而不是 DETECT 中。

---

## 必需输出：`## Stated Claims`

**每个符合条件的阶段必须发出 `## Stated Claims` 部分**，列出它所依赖的每个负载均衡声明，并附有以下四个标签之一。这个列表是接线知道将哪些声明路由到分类器的方式。

```markdown
## Stated Claims

- [KNOWN] <代理已通过源验证的声明，或是仓库内部明显的声明>
- [GUESSED:unknown] <代理完全缺少事实的声明 — 完全没有基础>
- [GUESSED:assumed] <代理断言了从未验证过的外部形状/值/行为的声明>
- [GUESSED:stale] <代理知道但可能已过时的外部/时间可变事实>
```

### 四个标签 — 精确语法

接线（D3/D4）机械解析此部分。使用 **精确** 这些标签（区分大小写，方括号语法）；其他格式被解析器忽略并视为未标记：

| 标签 | 含义 |
|-----|------|
| `[KNOWN]` | 你已经验证了这个声明：它是仓库内部明显的（grep、Read，或你此会话的工作上下文），或者你在本会话中引用了你获取的来源。 |
| `[GUESSED:unknown]` | 你完全缺少这个事实 — 你没有任何基础。 |
| `[GUESSED:assumed]` | 你断言了一个你从未验证过的形状、返回值、限制或行为。"合理"和"会有道理" **不是** `KNOWN`。 |
| `[GUESSED:stale]` | 你知道它，但它是外部/时间可变的事实（API、价格、模型 ID、库签名、速率限制），可能在你上次看到后已经改变。 |

### 三种猜测类型（auto-research-contract §1.3）

任何 `[GUESSED:*]` 声明都会触发 classify+verify 流水线。类型对可解释性很重要，但三种路由方式相同：

1. **Unknown** — 你没有任何基础。代理缺少这个事实。
2. **Assumed** — 自信但未验证。Binvoice S2-M5 事件中的主要失败模式（API 返回形状被作为事实陈述，从未检查）。**合理 ≠ 已确认。**
3. **Stale** — 曾经知道，但现在可能不是了。**默认: 趋向验证的失败。** 任何外部/时间可变的事实如果没有 **新鲜的引用来源**，就是过时的 → 研究。不要相信你对自身过时性的自我评估 — 这种自我评估本身又是另一种猜测。

### 诚实尽力合同

- 你 **必须** 尝试标记你所依赖的每个负载均衡声明。
- 一个你 **未能** 标记的声明是 **已确认的遗漏** — 一个 LLM 提示检测器的限制 — **不是** 你标记的声明的静默通过。
- 确定性保障（CLASSIFY + 通过 §7 标记的 ENFORCE）适用于每个 **已标记的** 猜测声明。DETECT 覆盖率是你的工作；执行是门控的工作。
- **缺失的（未声明的）缺口是尽力而为的遗漏，不是静默通过。** 你标记的越多，系统就越能代表你验证。

### 什么算"负载均衡"

如果此阶段中的交付物或决定 **依赖** 它正确，一个声明就是负载均衡的。示例：
- 你的代码将解析的外部 API 响应的形状。
- 版本或功能可用性声明（"此 SDK 支持 X"）。
- 你的逻辑需要尊重的速率限制或配额。
- 第三方工具的默认值或标志行为。
- 你正在实现的契约条款、规范措辞或标准要求。

内部声明（此仓库自己的代码、契约、测试、schema — 你可以通过 grep/Read 验证的东西）如果你在本会话中刚刚验证过，就是 `[KNOWN]`，不是 `[GUESSED]`。

## 对于接线（D3/D4 参考 — 非阶段代理）

接线迭代每个 `[GUESSED:*]` 行通过 `bin/gsd-t-research-gate.cjs`。每个外部猜测 → 研究阶段 → `## Verified Facts (auto-research)` 代码块 + §7 ENFORCE 标记。每个内部猜测 → grep/Read（如果 grep 返回空则升级到外部 — §5.1）。`## Stated Claims` 标题是可机器解析的入口点；接线精确 grep 此标题字符串。
